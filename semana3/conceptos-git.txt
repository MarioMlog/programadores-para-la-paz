Repositorio:
Un repositorio es el lugar donde se almacenan los archivos de un proyecto y todo su historial de cambios. Permite organizar el trabajo y facilitar la colaboración entre varias personas.

Commit:
Un commit es un registro de los cambios realizados en un proyecto en un momento determinado. Cada commit guarda una versión del trabajo para poder consultarla o recuperarla más adelante.

Push:
El comando push se utiliza para enviar los cambios guardados en el repositorio local hacia GitHub. De esta manera, el proyecto queda actualizado y disponible en la nube.
