El login permite identificar a un usuario antes de darle acceso a las funciones de una aplicación. 
Gracias a la autenticación es posible controlar quién puede ingresar y qué acciones puede realizar. 
Algunas rutas deben estar protegidas porque contienen información privada o permiten modificar datos importantes. 
De esta manera se evita el acceso no autorizado. 
No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
